# [API Management for the AI Era: Leveraging OpenAPI Standards](https://github.com/dret/api-days-workshop-paris-2025)

## Workshop at [API Days Paris](https://www.apidays.global/events/paris), December 10, 2025

### Final Quiz

Please keep in mind that some questions have more than one correct answer.

1. What changed APIs the most in the 2020s?

- ⬜ More mobile apps
- ☑️ AI becoming both API consumer and provider
- ⬜ JSON replacing XML
- ⬜ HTTP/2 becoming dominant

2. What is the main difference between DX and AX?

- ⬜ Developers use JSON, agents use XML
- ☑️ Developers interpret ambiguity; agents cannot
- ⬜ Agents write code, developers do not
- ⬜ DX is for REST, AX is for GraphQL

3. Which statement about OpenAPI descriptions is correct?

- ☑️ They are machine-readable contracts used by tools and agents
- ⬜ They are optional documentation add-ons
- ⬜ They exist only for public APIs
- ⬜ They replace API versioning

4. Which of the following improves both DX and AX?

- ☑️ Complete and meaningful summaries
- ☑️ Comprehensive examples
- ⬜ Removing error responses
- ⬜ Using only 200 OK for all operations

5. Why do AI agents struggle with many existing APIs?

- ⬜ Because agents cannot call HTTP
- ⬜ Because OpenAPI is too complex
- ☑️ Because APIs often have vague descriptions and inconsistent semantics
- ⬜ Because APIs require user interfaces

6. What is the purpose of API linting?

- ☑️ Ensure quality and consistency across APIs
- ⬜ Speed up server performance
- ⬜ Convert APIs from XML to JSON
- ⬜ Automatically generate business models

7. When should overlays be used?

- ☑️ To modify or enrich an OpenAPI document without changing the source file
- ☑️ To produce audience-specific variants (public/partner/internal)
- ⬜ To add new actual API endpoints
- ⬜ To migrate OpenAPI to AsyncAPI

8. What does Arazzo primarily describe?

- ☑️ Workflows composed of multiple API calls
- ⬜ JSON Schema validation rules
- ⬜ API authentication mechanisms
- ⬜ Code-generation templates

9. Why are examples important in OpenAPI?

- ☑️ They help developers learn usage patterns
- ☑️ They help agents plan safe API execution
- ⬜ They are required only for error responses
- ⬜ They replace schemas

10. Which HTTP method mapping is correct?

- ☑️ POST → create or trigger
- ⬜ GET → modify a resource
- ⬜ PUT → partially update a field
- ⬜ PATCH → delete a resource
